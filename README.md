# Portfolio Website

## Description
This is my web development practice project. It is a personal portfolio website where I showcase my projects, skills, and contact information.  
The website is built using **HTML** and **CSS**.

## Features
- Clean and responsive design
- Showcase of personal projects
- Contact section
- Smooth navigation

## How to Run
1. Download or clone this repository:  
   `git clone https://github.com/artikolhe3/portfolio-website.git`
2. Open `index.html` in your web browser.

## Skills Used
- HTML
- CSS
- Web design fundamentals
- Git and GitHub for version control
